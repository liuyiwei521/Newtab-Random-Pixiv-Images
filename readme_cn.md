# Newtab Random Pixiv Images

本扩展会从 Pixiv 搜索结果中随机选取一张图片作为新标签页背景。默认使用 "10000users入り" 等热门标签作为搜索关键词。右下角可以刷新、打开标签管理、查看作品页面并排除标签。（使用前可能需要先登录 Pixiv 以确保搜索功能正常）

## 功能特性

- 🎨 **随机壁纸** — 每次打开新标签页自动展示一张 Pixiv 插画
- 🔄 **一键刷新** — 右下角刷新按钮快速切换图片
- 🔎 **查看作品** — 眼睛图标打开当前作品页面
- 🏷️ **标签管理** — 可视化管理搜索关键词（AND / OR / 排除），支持预设
- 🚫 **全局屏蔽** — 统一屏蔽标签（对所有预设生效）
- 📦 **导入/导出** — 备份或共享 JSON 配置
- 🌐 **多语言** — 支持英文、中文、日文界面切换

## 安装指南
### chrome/edge:
1. 克隆或下载本项目到本地
2. 打开浏览器并访问 chrome://extensions/
3. 开启右上角的 开发者模式 开关
4. 点击 加载已解压的扩展程序 按钮，选择项目中的 src 目录
5. 打开新标签页

### firefox:
1. 克隆或下载本项目到本地
2. 打开浏览器并访问 about:debugging
3. 在左侧选择 This Firefox 标签
4. 点击 Load Temporary Add-on... 按钮，进入 src_firefox 目录选择 manifest.json 文件
5. 打开新标签页

## 使用方式

- **新标签页**：打开新标签页即可看到随机 Pixiv 插画背景。右下角有刷新按钮（🔄）和标签管理按钮（⚙️）。
- **标签管理**：点击新标签页右下角的 ⚙️ 按钮进入标签管理页面。

## Pixiv 搜索布尔逻辑

Pixiv 搜索支持丰富的布尔运算符进行标签过滤：

### 基础运算符

| 运算符  | 语法          | 示例                   | 说明               |
| ------- | ------------- | ---------------------- | ------------------ |
| **AND** | `A B`（空格） | `初音ミク VOCALOID`    | 两个标签都必须匹配 |
| **OR**  | `A OR B`      | `初音ミク OR 鏡音リン` | 匹配任一标签       |
| **NOT** | `-A`          | `-R-18 -漫画`          | 排除标签           |

### 括号分组

| 模式              | 语法                  | 示例                                                                 |
| ----------------- | --------------------- | -------------------------------------------------------------------- |
| **OR 分组**       | `(A OR B)`            | `(10000users入り OR 30000users入り)`                                 |
| **AND + OR 组合** | `A (B OR C)`          | `風景 (夕焼け OR 朝焼け)`                                            |
| **多个 OR 分组**  | `A (B OR C) (D OR E)` | `VOCALOID (初音ミク OR 鏡音リン) (10000users入り OR 30000users入り)` |

### 高级模式

| 模式               | 语法                      | 示例                                                    |
| ------------------ | ------------------------- | ------------------------------------------------------- |
| **嵌套括号**       | `((A OR B) OR (C D))`     | `((初音ミク OR 鏡音リン) OR (VOCALOID 女の子))`         |
| **OR 连接 AND 组** | `(A B) OR (C D)`          | `(初音ミク 10000users入り) OR (鏡音リン 5000users入り)` |
| **显式 AND 嵌套**  | `((A OR B) AND (C OR D))` | `((水彩 OR 油絵) AND (風景 OR 自然))`                   |

### 复杂示例

```
VOCALOID (初音ミク OR 鏡音リン) (10000users入り OR 30000users入り) -R-18 -漫画
```

该查询含义为：**VOCALOID** 且（**初音ミク** 或 **鏡音リン**）且（**10000users入り** 或 **30000users入り**），排除 **R-18** 和 **漫画**。
